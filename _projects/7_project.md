company-os is an AI-native operating layer for company workflows, designed to coordinate people, agents, and systems through shared execution logic instead of siloed tools.

**Key Capabilities:**
- **Agent Orchestration:** Coordinates specialized agents for planning, execution, review, and follow-up across business functions
- **Workflow Automation:** Converts repeatable SOP-style processes into structured, trackable automations with human checkpoints
- **Multi-Team Coordination:** Keeps handoffs explicit across teams so ownership, status, and next actions stay visible

**Architecture:**
- **Control Plane + Execution Plane:** A central orchestration layer manages policy and routing while worker agents run domain tasks
- **Shared State + Integrations:** Common context and system connectors keep decisions consistent across tools, teams, and timelines
