## Recent Open-Source Tools

- [slop-detect](https://github.com/vikasudasi/slop-detect) — Privacy-preserving CLI that detects AI-written text using 5 local heuristics
- [prompt-riser](https://github.com/vikasudasi/prompt-riser) — CLI that runs prompts through multiple reasoning strategies
- [pr-classify](https://github.com/vikasudasi/pr-classify) — Detects AI-generated PRs and generates audit reports
- [skill-audit](https://github.com/vikasudasi/skill-audit) — Audits and validates agent SKILL.md files
- [prompt-snapshot](https://github.com/vikasudasi/prompt-snapshot) — Snapshots project context for AI agent prompts
- [my-agent](https://github.com/vikasudasi/my-agent) — Personal macOS deep agent for your terminal
- [AI Task Management System](https://github.com/vikasudasi/my-agent-project) — Multi-agent coordination framework
